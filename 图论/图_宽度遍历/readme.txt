图的宽度搜索（Breadth-First-Search）
	
	简称BFS，是一种图形搜索算法。简单的说，BFS是从根节点开始，沿着树的宽度遍历树的节点。如果所有节点均被访问，则算法中止。
	
	图的存储结构采用 邻接表 的方式
	
	搜索过程中用到队列结构

v1 - v2
v1 - v3
v2 - v3
输出图的链接信息：
v1 ->2 ->1 
v2 ->2 ->0 
v3 ->1 ->0 
v4
