威佐夫博弈（Wythoff Game）

	有两堆若干个物品， 两个人轮流从某一堆或同时从两堆中取同样多的物品， 规定每次至少取一个，多者不限，最后取完者胜。
	
	1、可以用两个数(a[k], b[k], 且 a[k] <= b[k], k为自然数) 表示两堆物品的数量。如果该数量为奇异局势（必败局势）， 那么
	   先手输。
	2、奇异局势，前几项如下：(0,0) (1,2) (3,5) (4,7) (6, 10) ......
	   根据上面的数据它（奇异局势）具有如下性质
	   (1)、a[k]是在之前的奇异局势(a[k-1],b[k-1])中未出现过的最小自然数。
	   (2)、b[k] = a[k] + k, k代表出现的第几个奇异局势。
	   (3)、任何自然数都包含在一个且仅一个的奇异局势中。
	   (4)、任何操作（一次动作）都会将奇异局势变成非奇异局势。
	   (5)、可以采用适当的方法将非奇异局势变成奇异局势，那么下一个必输。
	   
	
	结论：两个人如果都采用正确的操作，那么面对非奇异局势， 先拿者必胜；反之，后拿者胜。
	
	判断一个局势(a, b)是不是奇异局势, 且 a < b, 有如下公式：a = (b - a) * 1.618
