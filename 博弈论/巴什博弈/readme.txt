巴什博弈（bash game）
	只有一堆n个物品，两个人轮流从这堆物品中取物品，规定每次至少取一个，最多取m个，最后取光者为胜。
	
	1、如果n = m + 1， 由于一次最多只能取 m 个， 所以， 无论先取者拿走多少个， 后取者都能够一次拿走剩余的物品， 后取者胜。
	
	2、根据（1）中的推理， 当 n = (m + 1) * r + s， （r为任意自然数， s =< m）， 如果先取者从 n 中拿走 s个物品，后取者拿走 k（=< m） 个， 那么先取者再拿走 m + 1 - k 个， 结果剩下 （m + 1） * （r - 1） 个， 以后保持这样的取法， 那么先取者是获胜者。 总之， 要保证给对手留下（m + 1）的倍数， 就能获得最后胜利。
	
	3、只要 n % (m + 1) != 0 先取者一定获胜。
