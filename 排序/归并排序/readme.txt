归并排序算法
	
	归并排序是建立在归并操作上的一种有效的排序算法，归并排序算法是采用分治法的一个典型应用。对一个给定的数表进行排序，首先把它们划分成相等的两半，递归地对每一半分别排序，然后组合这些递归调用的结果。对于初始划分与最后的重新组合仅使用线性时间。归并排序是一种较为占用内存的 稳定排序算法。
	
	假设有待排序的数组 M[n]， 其中 M[i] 和 M[j] 为数组中的两个元素， 且 M[i] 和 M[j]分属于不同的有序单元中， 同时， 分属于的两个有序单元 具有相同的调用函数（有序单元相邻）。
	归并过程：
		1、比较M[i] 和 M[j]的大小， 若 M[i] <= M[j]， 则将 M[i] 复制到 Buf[k] 中， 同时将i 和 k自加1；
			否则将 M[j] 复制到 Buf[k] 中， 同时将j 和 k自加1；
		2、按照规则1一直循环进行，直到一个有序单元中的数字取完，然后在将另一个有序单元中剩余的数字复制到Buf[k]
