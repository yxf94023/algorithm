问题：在二维平面上给出一些点的坐标， 在这些点中， 找出邻近距离最小的点对


分析：
为了问题的分析和描述， 定义一下符号
	P表示所有点的集合{p(1), p(2), p(3), ..., p(n)}
	P'为集合， 且属于集合P
	表P'(x) 表示集合 P' 中的点按照 x轴排序
	表P'(y) 表示集合 P' 中的点按照 y轴排序
	
	
步骤一、对属于集合P中的点按照x轴 和 y轴大小分别排序， 得到表T(x) 和 T(y)
步骤二、将集合中的点依照表T(x)拆分成两部分 [0, T(x)/2], 和 (T(x)/2, T(x)]， 分别用集合L 和 集合 R表示这两部分的点。
步骤三、对集合L 和 集合R，递归地执行步骤一 和 步骤二中的操作， 直到被划分的集合中包含的点数量不大于3，则停止递归。
步骤四、求最邻近的点对，最邻近的两点可能在的情况有，(1)两点都在集合L中； (2)两点都在集合R中；(3)两点中一个点在L 另一个点在 R中。  其中第(1)和第(2)情况中的不需要特殊处理直求2点间的距离即可，那么如何确定第(3)点中的情况是最邻近的点对呢？ 下面详细说明第(3)点：

	假设点p'(x', y')是集合L中 依照x轴排序 x轴上最大的点，在x'处与y轴平行地做一条垂直线(x = x')
	假设d(min)是从集合L 和 集合R 中获得的点对的最小距离
	
如果最小的邻近点对出现在情况(3)， 那么存在点l(x(l), y(l)) 和 点r(x(r),y(r))，点l属于集合L，点r属于集合R，它们的距离为d(l,r) < d(min)， 则点l，y属于以下范围的点S(y)， 在集合L和集合R中与垂线(x = x')相距d(min)距离的所有点，这些点按照y轴方向排序。

此处省略一些说明（没有图不好描述了）， 每一个属于表S(y)中同时属于集合L中的点只需要计算8个连续的单位(单位长度为d(min)/2)，即可求得最邻近的点对。
